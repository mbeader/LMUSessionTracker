# Web

The LMUSessionTracker Angular front-end.

## Build

Run `ng build`.
Specifying `--configuration development` will do a dev build.
Static builds (with in browser WebAssembly Sqlite database) can be made using `--configuration static`.

## Test

Run `ng test`.
Coverage is included in the main `coverage.bat` script.

## Deploy

Deployment to GitHub Pages can be done by setting the env var `GH_TOKEN` and running `ng deploy`.
